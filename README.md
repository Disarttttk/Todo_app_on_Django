# To Do List

Web-приложение для хранения заметок на Django

## Features

- Хранение пользовательских заметок в БД
- Добавление и удаление заметок, статус заметки
- Регистрация и авторизация пользователя 
- Личный кабинет
- Админ панель

## Technologies

- Python
- Django
- MySQL
- Unittest
- JavaScript
- Ajax
- CSS
- HTML
